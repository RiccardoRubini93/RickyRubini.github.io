Sparsification techniques for Reduced Order Models of turbulent flows

The main purpose of this page is to give a first glimpse on the thematic I am tackling in my PhD project... This is still an ongoing research so the page is not complete. New contents will be uploaded soon!

Motivation

In the classical description of developed turbulent flows energy is transferred across the hierarchy of coherent structures via nonlinear triadic interactions. Implicit in this picture is the fact that not all interactions have the same importance, but they occur in preferential patterns. Evidence points out that the nonlinear interaction pattern among coherent structures is sparse

Mathematically we can visualise each structure as a point in a graph and each interactions as an edge connecting two different points. Therefore, if the two structures are interacting a non-zero value in the matrix that describes the graph of the energetic interactions is present.

The last sentence translated in the Machine Learning framework where a structure is represented as a dimension of the system sums up to : you need a non-zero value when two dimensions interacts between each others.

Therefore, you should ask yourself... Are all the dimensions of my input vector interacting with all the others?

Methodology

Reduced Order Modelling

A successful approach to tackle high complexity flow is to decompose the field into spatial and temporal modes that satisfy certain properties. Then, as displayed in figure (1) the governing equation are projected onto the subspace spanned by a subset of these modes. As a result the original system is equivalent to a set of coupled ODEs. As shown in the last panel of figure (1) the system is composed by a constant, linear and quadratic term corresponding to the pressure, viscous and convective term in the Navier-Stokes equations. A popular approach is projecting the governing equations onto the subspace spanned by the POD modes.

The main limitation of this approach arises from the non linear nature of the system where every mode interacts with all the others in a non trivial way. This makes the tensor $\color{black}{Q_{ijk}}$ dense producing computationally expensive models with an hindering the underlying physics.

Our main goal is to integrate the machine learning framework in the process of generating the Reduced Order Model and learning from data which terms are truly relevant for the dynamics. We believe that this could be the key enabler to obtain a family of computationally efficient and easily interpretable models that contains only the necessary physics.

Sparsification of ROMs

In a nutshell machine learning is the field of computer science that "uses statistical techniques that give the machine the ability to "learn" from data without being explicitly programmed. Figure (2) shows an overview on the possible machine learning techniques. The proposed approach aims to couple modal decomposition techniques and regression technique to automatically generate from data models where only the important features of turbulence are retained

It is known that the energy transfer inside a turbulent flows occurs according preferential patterns. Therefore not all triadic interaction have the same importance. The dynamics of a certain scale depend predominantly upon the dynamics of a subset of the set of scales and the complementary set of structures can be generally neglected with minor global effects

Figure 3 (a) shows how our idea can be located in between a pure Machine Learning approach (very effective with complex systems but poorly generalise) and a first principle approach (very general but difficult and expensive to use in case of complex physics). We aim to bridge the gap between this two diametrically opposite approaches.

Figure 3 (b) shows a sketch of the step needed to set up our framework. They can be summarized as follow.

  1. Fully resolved simulation to generate the data-set
  2. Projection of the governing equations onto an optimal subspace generated by modal decomposition as shown in figure (1)
  3. The reduced order model can be rearranged in a matrix form suitable to perform regression. In addition, to preserve the physics of the original system we add a linear relation between the coefficients. In this case, we ensure that the nonlinear term in the system is energy preserving as the original nonlinear term in the Navier-Stokes equations.
  4. We solve the regression problem with a $\color{black}{l_1}$ penalisation of the coefficients. The $\color{black}{l_1}$ norm encourage the sparsity of the solutions retaining only the relevant terms of the system.
  5. A whole family of sparse systems has been generated that can be used for analysis or flow control

Two dimensional turbulence

As demonstration, we applyed this methodology to the unsteady flow developing inside a lid driven cavity at Re = 10.000. Panel (4) shows an animation of the magnitude of the velocity field. We observe that, despite the simple geometry, the flow field presents complex flow features such recirculation zones in the corners of the cavity and a strong shear layer. These features make this problem a good test bed for the proposed methodology.

The sparse model is obtained solving the LASSO problem

$$\color{black}{ \hat{{\boldsymbol \beta}}_i = \underset{{\boldsymbol \beta}_i}{\mathrm{argmin}} \left \{ ||{\boldsymbol \Theta}(A) {\boldsymbol \beta}_i - \dot{A}_i||_2^2 + \gamma||{\boldsymbol \beta}_i||_1 \right \}, \quad \quad i=1,...,N}$$

This is composed by a least-squares problem that minimise the error of the reconstructed solution with respect the target and a $\color{black}{l_1}$ penalisation term that encourages the sparsity of the corresponding solution. The relative importance between this two terms is weighted by the regularisation weight indicated as $\color{black}{\gamma}$. It is not possible to set a priori the value of the regularisation weight so we varied during the analysis generating a family of models with different sparsity/accuracy trade-off and plotting them model on a sparsity accuracy plane. Ideally, the optimal model is the one with low density (corresponding to a reduced number of coefficients) and low prediction accuracy.

Figure (5) shows the sparsification obtained solving the LASSO problem for increasing values of the regularisation weight. Interestingly we observe the presence of a large flat part for high value of the density of system. This means that is possible to eliminate many terms from the system without affecting the reconstruction error. In addition, we can identify an optimally sparse system, in this case retaining only 15% of the coefficients ($\color{black}{\rho \sim 0.15}$), that present low density and low value of the reconstruction error.

We now analyse the performance of a optimally sparse model obtained with a value of the regularisation weight that we will indicate as optimal $\color{black}{\gamma}$.

Panel (6) shows the comparison between the sparse and the dense ROM displaying the temporal evolution of the turbulent kinetic energy defined as:

$$\color{black}{E(t) = \frac{1}{2}\int_{\Omega} \mathbf{u}^{\prime}(t,\mathbf{x})^2 d\Omega}$$

The dense model has been generated applying model order reduction (Figure 1) to the training dataset while the sparse model is generated applying the procedure explained in figure (3) on the same dataset. We observe that the sparsification procedure introducing physical constraints in the model is able to correctly estimate the level of turbulent kinetic energy in the system. On the other hand, the dense model obtained solely modal decomposition and Galerkin projection tends to overshoot its value. To conclude this example panel (7) shows the result of the time integration of the sparse system. From visual inspection we observe that the optimally sparse model reproduces the flow features of the original DNS with a reduced number of interactions.

Ongoing Work

The last part of the research will be dedicated to two works in parallel. The first is extend and further validate the framework presented for the two cavity flow to a three dimensional test case. The second is the development of the Sparse Interacting Decomposition (SID).

Three Dimensional Turbulence

We are applying our framework to a three dimensional turbulent flow where the interactions between flow structure are local. The main target of this analysis is to show if and how it is possible to exploit the locality of the interactions to generated more and more sparse models as the Reynolds number increases but keeping, at the same time, an approximately constant reconstruction error

Sparse Interacting Decomposition SID

We observe that the physics of a turbulent flow is intrinsically sparse since a given structure interacts only with a small subset of all the other structures. However, classical reduction order techniques are not able to incorporate this sparsity. We are able to restore this sparsity with a posteriori procedure performed on the dense ROM. The main idea behind the SID decomposition is to generate a modal decomposition whose spatial modes incorporate a priori the sparsity of turbulent flows generating then a ROM that present already sparsity features.

With this a priori methodology I'm currently analysing the separated flow over a backward facing step.

Interestingly, this open flows has a strong advective nature displaying strong vortex shedding phenomena. The idea consists in finding out if the a priori sparsification can identify interesting flow structures.
