Incompressible Navier Stokes solver

  • 2-D Navier Stokes incompressible solver, the algorithm implemented is a Pressure Based SIMPLE algorithm. To compute the pressure I used the Poisson equation technique of the artificial compressibility approach. The code solves the system of 2-D Navier Stokes equations coupled with the equation of transport of a non interacting scalar variables (Temperature in this case). The algorithm is applied to a very simple cavity flow but it is general for every geometry on a structured mesh. Finally I would like to thank Lorena A. Barba Group that with her tutorials helped me a lot to understand how to structure and implement the code. The code is also avaible in THIS Git Hub repo.

DyNN

Small sofware package that teaches a Deep Neural Network how to learn a non-linear dynamical system, Lorenz system in this case. At an high level this procedure can be summarised in three steps

(1): Create a training set integrating the Lorenz system from N = 10 randomly chosen initial conditions.

(2) Train the NN feeding the input and the output layer with the time evolution at t and t+dt, respectively.

(3): Once the NN is trained one can use to integrate the system of dt obtaining a dynamics that evolves in the same phase space region of the true one.

The application to the Lorenz system is trivial, however this methodology could have interesting applications where the non-linear dynamics of the system is not known.
